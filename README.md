# youtube-rag-chatbot
An AI-powered YouTube RAG chatbot built using LangChain, Hugging Face, Chroma Vector Database, and Streamlit. The application extracts YouTube transcripts, creates embeddings, performs semantic search, and answers questions using Retrieval Augmented Generation (RAG).
